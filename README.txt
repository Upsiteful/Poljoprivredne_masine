STATIČKI SAJT - UPUTSTVO

Fajlovi:
- index.html                početna stranica
- kontakt.html              kontakt strana sa formom i Google mapom
- masine.html               pregled kategorija mašina
- masine-kategorija.html    prikaz modela u jednoj kategoriji mašina
- delovi.html               prvi nivo delova (za koju mašinu)
- delovi-pregled.html       drugi nivo delova (koji deo)
- proizvodjaci.html         treći nivo delova (proizvođači)
- pretraga.html             rezultati pretrage sa početne strane

VAŽNO ZA EMAILJS:
U kontakt.html zamenite sledeće vrednosti:
- UBACI_PUBLIC_KEY
- UBACI_SERVICE_ID
- UBACI_TEMPLATE_ID

Google mapa trenutno koristi primer lokacije. Zamenite iframe URL stvarnom lokacijom.

Telefon, mejl i Facebook link su trenutno demo podaci i mogu lako da se promene u svakom fajlu.

Sve stranice imaju ugrađen CSS i JS unutar HTML fajlova, kako je traženo.
